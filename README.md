# Mercadolibre Price Monitoring
This app monitors the price of certain technology products in Argentina in local currancy, and compares it to USD in real time.

## Tech Stack
- **Backend:** Java 17, Spring Boot, PostgreSQL.
- **Data:** Python (Pandas, Requests).
- **Frontend:** React + TypeScript.
- **Infra:** Docker.
